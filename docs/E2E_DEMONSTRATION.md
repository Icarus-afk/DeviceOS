### Module Responsibilities

| Module | Responsibility |
|--------|---------------|
| **devices** | Device registration, CRUD, secret key generation |
| **telemetry** | Ingest, query, latest datapoint, WebSocket broadcast |
| **alerts** | Rule-based alerting with threshold evaluation |
| **auth** | JWT tokens, API key auth, device auth middleware |
| **commands** | Send remote commands, track execution results |
| **ota** | Firmware upload, deploy, rollout tracking |
| **webhooks** | Outbound event delivery with retry tracking |
| **fleet** | Device groups, tags, fleet health aggregation |
| **tenant** | Multi-org management, user invitations |
| **audit** | Immutable audit log for all operations |
| **simulator** | Built-in virtual device generator for testing |
| **dashboard** | Single-page web UI with real-time WebSocket |
